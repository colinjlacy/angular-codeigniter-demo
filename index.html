Angular/CodeIgniter Demo

A simple demonstration using AngularJS to make AJAX calls to a CodeIgniter backend.

Push the button below to spring Angular into action!

Each word from the following phrase has been entered as a row in the database:

"The cake is a lie!"

Each time you push the button, Angular will make a request to CodeIgniter, which will pull one row at random, and send it to Angular.
